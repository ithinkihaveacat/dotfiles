# Tools

This directory contains various utility tools.

## Installation & Updates

To build and install all tools available in this directory, simply run the
update script from the root of the repository:

```bash
./update
```

This script will:

1. Build each tool.
2. Install the binaries to `~/.local/bin/`.

## Development & Usage

For specific usage instructions, development guides, or troubleshooting for a
particular tool, please refer to the `README.md` file within that tool's
subdirectory.
